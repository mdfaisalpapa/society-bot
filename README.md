# Society Bot - Telegram Bot for Property Management

## Features

- Tenant Management
- Visitor Management  
- Maintenance Tickets
- Pet Registration

## Installation

```bash
git clone https://github.com/mdfaisalpapa/society-bot.git
cd society-bot
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
python app.py
```

## Configuration

Edit `.env` with your settings.
